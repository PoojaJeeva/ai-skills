# ai-skills

A personal collection of AI skill definitions and MCP (Model Context Protocol) servers.

## Structure

```
ai-skills/
├── skills/        # Skill definition files (.md)
├── mcps/          # MCP server projects
└── README.md
```

## Skills

The `skills/` directory contains `.md` files that define reusable AI skills — prompts, workflows, and techniques for working effectively with AI models.

## MCPs

The `mcps/` directory contains MCP server implementations that extend AI assistants with custom tools and integrations.
